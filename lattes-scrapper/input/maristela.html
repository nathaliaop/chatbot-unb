Uma Arquitetura para Acesso a Dados Geográficos na Web e em um Ambiente de Computação Móvel
Descrição: Banco de Dados geográficos são aqueles que possuem uma dimensão espacial, ou uma localização, representando o mundo geográfico real como as imagens de satélites de sensoriamento remoto, os dados de inventários cadastrais, os dados ambientais, podendo ser apresentado por mapas, imagem digital, representação de rede de transporte, dentre outros. Atualmente, existem banco de dados geográficos espalhados em diferentes lugares, sendo desenvolvidos por diferentes grupos de pesquisa. Um dos grandes desafios nesse contexto é a integração desses dados heterogêneos e distribuídos, mas que podem estar representando dados de um mesmo espaço geográfico. Um outro desafio é a disponibilização desse dado geográfico através de mapas interativos que podem estar em dispositivos móveis ou em sistemas na web. Nesse contexto, esse projeto tem como objetivo especificar e implementar um framework de serviços que dará suporte as características de um SGBD (Sistema Gerenciador de Banco de Dados) de maneira integrada, tais como, armazenamento de dados, processamento de consulta e gerenciamento de transação, que poderá ser acessado tanto por um dispositivo móvel, como também em um sistema de informação geográfico na web..
